# Prepare fp object for fitting and prepare likelihood data

Prepare fp object for fitting and prepare likelihood data

## Usage

``` r
prep_fp_fitmod(obj, ..., epp = FALSE)
```

## Arguments

- obj:

  specfp or eppfp object

- ...:

  Additional parameters to override on \`obj\`

- epp:

  If TRUE prep EPP model fit

## Value

List containing \`fp\` object ready for fitting and likelihood data
\`likdat\`
