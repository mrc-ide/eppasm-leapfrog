# Prepare direct incidence input EPP simulation

Prepare direct incidence input EPP simulation

## Usage

``` r
prepare_directincid(pjnz, method = "directincid_hts")
```

## Arguments

- method:

  Method for intercalating new infections. Either "directincid_ann" for
  annual new infections or "directincid_hts" for new infections each HIV
  time step.
