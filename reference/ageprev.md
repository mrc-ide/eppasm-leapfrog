# Prevalence by arbitrary age groups

Prevalence by arbitrary age groups

## Usage

``` r
ageprev(
  mod,
  aidx = NULL,
  sidx = NULL,
  yidx = NULL,
  agspan = 5,
  expand = FALSE,
  VERSION = "C"
)
```

## Arguments

- sidx:

  sex (1 = Male, 2 = Female, 0 = Both) Notes: Assumes that AGE_START is
  15 and single year of age.
