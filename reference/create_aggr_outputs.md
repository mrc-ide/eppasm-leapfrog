# Crate aggr outputs

Crate aggr outputs

## Usage

``` r
create_aggr_outputs(fitlist, fitnat, both = TRUE)
```

## Arguments

- fitlist:

  list of model fits to aggregate.

- fitnat:

  single fit to use as the base fit with aggregation.

- both:

  flag indicating whether to return outputs for both aggregated fit and
  individual fits (default TRUE).
